## 🔧 已知问题修复

### 高优先级（必须修复）
- [ ] **0.0秒耗时问题**
  - 原因：任务提交耗时太短，精度不足
  - 修复：设置最小统计粒度（0.001秒）
  - 验证：统计值 > 0 或显示 "<0.001秒"

- [ ] **"间隔过短"误报**
  - 原因：测试/快速操作被误判为异常
  - 修复：添加去抖窗口（5秒）+ 白名单
  - 验证：正常快速操作不触发警告

- [ ] **JSONL 重复追加**
  - 原因：每次更新都追加，导致重复
  - 修复：定期压缩（保留最新）
  - 验证：文件大小不无限增长

### 中优先级（建议修复）
- [ ] 置信度计算优化（避免除零）
- [ ] 错误信息国际化（中英文）
- [ ] 日志级别控制（debug/info/warning/error）

### 低优先级（可延后）
- [ ] Dashboard 集成
- [ ] 可视化图表
- [ ] 导出/导入学习数据
